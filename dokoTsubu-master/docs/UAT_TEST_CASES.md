# ShiftFlow 受入テストケース

## 実施条件

- 対象: リリース候補版のShiftFlow
- 役割: 従業員、店長、人事担当者、店長の代理者
- 端末: 対応PCブラウザ、iPhone標準ブラウザ、Android Chrome
- 証跡: 実施日時、実施者、環境、結果、画面キャプチャまたは出力ファイル、不具合番号を各ケースに記録する
- 判定: 期待結果をすべて満たした場合だけ合格とする

| UAT-AC-17 | 5年間保存 | 保存期間境界の直前・当日・超過データをシフト、勤怠、有休、監査、位置情報に用意して保守処理を実行する。 | 5年以内は保持され、超過データは定義どおり削除または匿名化され、処理記録が残る。 | `DataRetentionServiceTest`、保守処理ログ |
